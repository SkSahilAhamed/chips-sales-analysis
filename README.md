# Chips Sales Analysis

## 🎯 Objective
The goal of this project is to analyze customer purchasing behavior and identify key drivers of sales performance using retail transaction data.

## 📊 Project Overview
This project analyzes transactional chips sales data to uncover sales trends, customer segments, brand performance, and product pack-size patterns. The project includes Python-based analysis and SQL queries for business KPI reporting.

## 🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Excel

## 🔍 Key Findings
- Premium customers showed higher average spend per transaction compared to budget customers
- A small number of brands contributed the majority of total sales
- Mid-sized pack products were purchased more frequently
- Monthly sales trends showed consistent demand patterns
- Customer segment analysis helped identify high-value customer groups

## SQL Analysis Included
This project includes SQL queries for business KPI reporting and revenue analysis.

Queries added for:
- Total revenue
- Total transactions
- Unique customers
- Monthly sales trend
- Top 10 products by revenue
- Top 10 brands by revenue
- Pack size performance
- Customer segment performance
- Revenue contribution percentage by brand
- Brand ranking using window functions
